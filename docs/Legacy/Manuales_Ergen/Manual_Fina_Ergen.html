Instalación y Puesta en Marcha

Fina Ergen está diseñada actualmente para entornos Linux (testeada en Ubuntu/Debian con Wayland/X11).

Requisitos Previos

  • Python 3.10 o superior
  • Conexión a Internet (para APIs de clima, noticias y updates)
  • Micrófono y Altavoces configurados

Pasos de Instalación

1. Clona el repositorio o descomprime el paquete en tu carpeta personal.

cd Fina-Phoenix
./install.sh

2. Configura tus dispositivos y claves:

cp config_template.py config.py
nano config.py  # Añade tus API Keys (OpenAI, NewsAPI, etc)

3. Inicia a Fina:

python3 main.py

Cómo Usar a Fina

Interactuar es natural. Solo di su nombre o usa el panel visual.

Palabra Clave: Di "Fina" claro y fuerte para despertarla. El anillo visual cambiará a color verde.

Comandos Esenciales

  • Información: "¿Qué hora es?", "¿Qué tiempo hace?", "Cuéntame las noticias".
  • Sistema: "Sube el volumen", "Abre Firefox", "Toma una captura".
  • Identidad: "¿Quién eres?", "Acerca de ti".
  • Productividad: "Recuérdame comprar leche a las 18:00", "Lee mi correo".
