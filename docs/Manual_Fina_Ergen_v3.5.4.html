El Corazón del Sistema

La nueva arquitectura permite que Fina sea instalada como un paquete nativo del sistema operativo, detectando automáticamente sus dependencias y rutas de ejecución sin configuración manual.

Esquema técnico de integración entre la UI, el puente Rust y el motor de IA Python.

Instalación Profesional

Fina ahora se distribuye en formatos binarios universales para distros Linux modernas:

# Instalación en Ubuntu/Debian/Mint
sudo dpkg -i Fina-Ergen_3.5.4-14_amd64.deb

# Ejecución Portable (AppImage)
chmod +x Fina-Ergen_3.5.4-14_amd64.AppImage
./Fina-Ergen_3.5.4-14_amd64.AppImage

Dinámica de Plugins Market

El control de equipos Samsung, LG, TCL, aires acondicionados Midea/Surrey y timbres Tuya ya no forma parte del peso base. Esto garantiza que Fina sea liviana y solo use la RAM necesaria para lo que tú tienes instalado.

Ruta de Usuario Protegida

Tus plugins, configuraciones y biometría se guardan permanentemente en:

~/.config/Fina/plugins/

Simplemente descarga la extensión desde el Fina Market y arrastra la carpeta allí. Fina la activará en el próximo inicio.

Diagnóstico y Mantenimiento

Al correr como una aplicación empaquetada, los errores ya no se ven en la consola. Fina ahora audita todo su comportamiento en archivos de texto para tu revisión.

Rastreo de Fallos (Logs)

  • Arranque Binario: ~/.config/Fina/fina_startup.log
  • Motores de IA: ~/.config/Fina/fina_services.log
